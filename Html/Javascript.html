¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se utiliza para hacer que las páginas web sean interactivas y dinámicas. Se usa para agregar funcionalidades como: Actualizar fuentes de redes sociales, Mostrar animaciones y mapas interactivos, Validar formularios, Actualizar contenido en tiempo real.

JavaScript se ejecuta directamente en el navegador del usuario, por lo que no es necesario recargar la página para realizar estas tareas.

Brendan Eich creó JavaScript en 1995 para agregar programas a páginas web en el navegador Netscape Navigator.

¿Cómo utilizar JavaScript con HTML?

Puedes hacerlo utilizando la etiqueta que debe envolver todo el código JavaScript que escribas. Se puede agregar el código JavaScript:

  • entre las etiquetas <head>
  • entre las etiquetas <body>
