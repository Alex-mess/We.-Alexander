Etiquetas de documento

Las etiquetas de HTML son códigos que se utilizan para estructurar y definir el contenido de un documento HTML. Estas etiquetas se encuentran en el Lenguaje de Marcado de Hipertexto (HTML) de cada página web.

Las etiquetas HTML están compuestas por: El carácter "<" (menor que), Una palabra o carácter que determina la etiqueta, Atributos HTML escritos de la forma "nombre =”valor”, El carácter ">" (mayor que).

html

La etiqueta <html> representa la raíz de un documento HTML.

La etiqueta <html> es el contenedor de todos los demás elementos HTML (excepto la etiqueta <!DOCTYPE>).

head

El elemento <head> es un contenedor de metadatos (datos sobre datos) y se coloca entre la etiqueta <html> y la etiqueta <body>.

body

La etiqueta <body> define el cuerpo del documento.

El elemento <body>> contiene todo el contenido de un documento HTML, como encabezados, párrafos, imágenes, hipervínculos, tablas, listas, etc.

Nota: Solo puede haber un elemento <body> en un documento HTML.

Etiquetas de Meta Datos

Las etiquetas meta, también conocidas como metadatos, son fragmentos de código HTML que se utilizan para describir el contenido de una página web. Se colocan en el encabezado del código fuente, entre las etiquetas HTML <html> y <body>.

Los metadatos son invisibles para el usuario, pero son muy útiles para los navegadores y los motores de búsqueda. Ayudan a los motores de búsqueda a entender el contenido de la página y a ofrecer los resultados más relevantes a los usuarios.

meta

La etiqueta <meta> define metadatos sobre un documento HTML. Los metadatos son datos (información) sobre datos.

Las etiquetas <meta> siempre van dentro del elemento <head> y normalmente se utilizan para especificar el conjunto de caracteres, la descripción de la página, las palabras clave, el autor del documento y la configuración de la ventana gráfica.

Los metadatos no se mostrarán en la página, pero la máquina podrá analizarlos.

Los metadatos son utilizados por los navegadores (cómo mostrar contenido o recargar la página), motores de búsqueda (palabras clave) y otros servicios web.

style

La etiqueta <style> se utiliza para definir información de estilo (CSS) para un documento.

Dentro del elemento <style> se especifica cómo deben representarse los elementos HTML en un navegador.

El elemento <style> debe estar incluido dentro de la sección <head> del documento.

title

La etiqueta <title> define el título del documento. El título debe ser solo texto y se muestra en la barra de título del navegador o en la pestaña de la página.

¡La etiqueta <title> es obligatoria en los documentos HTML!

El contenido del título de una página es muy importante para la optimización de motores de búsqueda (SEO). Los algoritmos de los motores de búsqueda utilizan el título de la página para decidir el orden en el que aparecen las páginas en los resultados de búsqueda.

base

La etiqueta <base> especifica la URL base y/o el destino de todas las URL relativas en un documento.

La etiqueta <base> debe tener presente un atributo href o un atributo target, o ambos.

Sólo puede haber un único elemento <base> en un documento y debe estar dentro del elemento <head>.

link

La etiqueta <link> define la relación entre el documento actual y un recurso externo.

La etiqueta <link> se utiliza con mayor frecuencia para vincular a hojas de estilo externas o para agregar un favicono a su sitio web.

El elemento <link> es un elemento vacío, solo contiene atributos.

Etiquetas de Agrupacion

Las etiquetas de agrupación en HTML son códigos que permiten organizar el contenido de una página web en grupos, como párrafos, listas, secciones, entre otros. También se pueden usar para dividir grupos similares con un efecto sutil, como una línea.

p

La etiqueta <p> define un párrafo.

Los navegadores agregan automáticamente una sola línea en blanco antes y después de cada <p> elemento.

Consejo: utilice CSS para dar estilo a los párrafos .

hr

La etiqueta <hr> define un cambio temático en una página HTML (por ejemplo, un cambio de tema).

El elemento <hr> se muestra con mayor frecuencia como una regla horizontal que se utiliza para separar el contenido (o definir un cambio) en una página HTML.

ul

La etiqueta <ul> define una lista desordenada (con viñetas). Utilice la etiqueta <ul> junto con la etiqueta <li> para crear listas desordenadas.

Consejo: utilice CSS para dar estilo a las listas .

Consejo: para listas ordenadas, utilice la etiqueta <ol>.

div

La etiqueta <div> define una división o una sección en un documento HTML. La etiqueta <div> se utiliza como contenedor de elementos HTML, a los que luego se les aplica estilo con CSS o se manipulan con JavaScript.

La etiqueta <div> se puede diseñar fácilmente utilizando el atributo de clase o id.
¡Se puede poner cualquier tipo de contenido dentro de la etiqueta <div>!

Nota: De forma predeterminada, los navegadores siempre colocan un salto de línea antes y después del elemento <div>.

figure

La etiqueta <figure> especifica contenido autónomo, como ilustraciones, diagramas, fotografías, listados de códigos, etc.

Si bien el contenido del elemento <figure> está relacionado con el flujo principal, su posición es independiente del flujo principal y, si se elimina, no debería afectar el flujo del documento.

figcaption

La etiqueta <figcaption> define un título para un elemento <figure>.

El elemento <figcaption> se puede colocar como el primer o último hijo del elemento <figure>.

Etiquetas Textuales

Las etiquetas de texto HTML son códigos que se usan para dar formato al texto de una página web, como ponerlo en negrita, cursiva o subrayado.

a

La etiqueta <a> define un hipervínculo, que se utiliza para vincular de una página a otra.

El atributo más importante del elemento <a> es el atributo href, que indica el destino del enlace.

De forma predeterminada, los enlaces aparecerán de la siguiente manera en todos los navegadores:

  • Un enlace no visitado está subrayado y en azul.
  • Un enlace visitado está subrayado y en color morado.
  • Un enlace activo está subrayado y en rojo.

span

La etiqueta <span> es un contenedor en línea que se utiliza para marcar una parte de un texto o una parte de un documento.

La etiqueta <span> se puede diseñar fácilmente con CSS o manipular con JavaScript usando el atributo de clase o id.

La etiqueta <span> es muy parecida al elemento <div>, pero <div> es un elemento de nivel de bloque y <span> es un elemento en línea.

sub

La etiqueta <sub> define el texto en subíndice. El texto en subíndice aparece medio carácter por debajo de la línea normal y, a veces, se representa en una fuente más pequeña. El texto en subíndice se puede utilizar para fórmulas químicas, como H2O.

sup

La etiqueta <sup> define texto en superíndice. El texto en superíndice aparece medio carácter por encima de la línea normal y, a veces, se representa en una fuente más pequeña. El texto en superíndice se puede utilizar para notas al pie, como WWW[1].

cite

La etiqueta <cite> define el título de una obra creativa (por ejemplo, un libro, un poema, una canción, una película, una pintura, una escultura, etc.).

Nota: El nombre de una persona no es el título de una obra.

El texto del elemento <cite> normalmente se muestra en cursiva.

Etiquetas Semanticas

Las etiquetas semánticas de HTML son elementos de código que ayudan a estructurar una página web y a definir el significado de su contenido.

nav

La etiqueta <nav> define un conjunto de enlaces de navegación.

Tenga en cuenta que NO todos los enlaces de un documento deben estar dentro de un elemento <nav>. El elemento <nav> está destinado únicamente a los bloques principales de enlaces de navegación.

Los navegadores, como los lectores de pantalla para usuarios discapacitados, pueden usar este elemento para determinar si deben omitir la representación inicial de este contenido.

section

La etiqueta <section> define una sección en un documento.

article

La etiqueta <article> especifica contenido independiente y autónomo. Un artículo debe tener sentido por sí mismo y debe ser posible distribuirlo independientemente del resto del sitio.

Fuentes potenciales del elemento <article>:

  • Publicación en el foro
  • Entrada de blog
  • Noticia

header

El elemento <header> representa un contenedor para contenido introductorio o un conjunto de enlaces de navegación.

Un elemento <header> normalmente contiene:

  • uno o más elementos de encabezado (<h1> - <h6>)
  • logotipo o icono
  • información de autoría

Nota: Puede haber varios elementos <header> en un documento HTML, pero <header> no se pueden colocar dentro de un <footer>, <address> u otro elemento <header>

main

La etiqueta <main> especifica el contenido principal de un documento.

El contenido dentro del elemento <main> debe ser exclusivo del documento. No debe contener ningún contenido que se repita en los documentos, como barras laterales, enlaces de navegación, información de derechos de autor, logotipos del sitio y formularios de búsqueda.

Nota: No debe haber más de un elemento <main> en un documento. El elemento <main> NO debe ser descendiente de un elemento <article>, <aside>, <footer>, <header> o <nav>.

footer

La etiqueta <footer> define un pie de página para un documento o sección.

Un elemento <footer> normalmente contiene:

  • información de autoría
  • información de derechos de autor
  • Información del contacto
  • Mapa del sitio
  • volver a los enlaces superiores
  • Documentos relacionados

Puedes tener varios elementos <footer> en un documento.
