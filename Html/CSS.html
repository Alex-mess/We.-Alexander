¿Qué es CSS?

CSS son las siglas de Cascading Style Sheets, que en español significa "Hojas de Estilo en Cascada". Es un lenguaje de diseño gráfico que permite controlar cómo se muestran los elementos en una página web, como su color, fuente, disposición, animaciones y efectos visuales.

CSS se utiliza junto con el lenguaje HTML, que se encarga del contenido básico de los sitios web. CSS separa el contenido del diseño, lo que facilita la gestión del aspecto visual de las páginas.

¿Cómo utilizar CSS con HTML?

Para utilizar CSS con HTML, puedes hacerlo de varias formas:

  • Hojas de estilo internas: El código CSS se coloca dentro del documento HTML, ya sea al principio o en el código fuente. Para hacerlo, se puede:
    1. Abrir la página HTML y localizar la etiqueta de apertura <head>
    2. Después de la etiqueta <head>, colocar el código <style type="text/css">
    3. Agregar las reglas CSS en una nueva línea
    4. Escribir la etiqueta de cierre: </style>
  • Hojas de estilo externas: El código CSS se escribe en un archivo separado con la extensión .css y luego se vincula al documento HTML. Para hacerlo, se puede:
    1. Crear un nuevo archivo en un editor de texto con la extensión .css
    2. Escribir las reglas CSS en el archivo
    3. Vincular el archivo CSS al documento HTML usando una etiqueta de enlace en la cabecera del documento
  • Estilos en línea: El código CSS se escribe en un atributo HTML style en una etiqueta

CSS interno

El CSS interno o incrustado requiere que añadas la etiqueta <style> en la sección <head> de tu documento HTML.

Este estilo CSS es un método eficaz para dar estilo a una sola página. Sin embargo, utilizar este estilo para varias páginas requiere mucho tiempo, ya que es necesario poner reglas CSS en cada página de tu sitio web.

CSS Externo

Con el CSS externo, enlazarás tus páginas web a un archivo .css externo, que puede ser creado con cualquier editor de texto de tu dispositivo (por ejemplo, Notepad++).

Este tipo de CSS es un método más eficiente, especialmente para estilizar un sitio web grande. Al editar un archivo .css, puedes cambiar todo tu sitio de una sola vez.

CSS en línea

El CSS en línea se utiliza para dar estilo a un elemento HTML específico. Para este tipo de estilo CSS, sólo tendrás que añadir el atributo style a cada etiqueta HTML, sin utilizar selectores.

Este tipo de CSS no es realmente recomendable, ya que cada etiqueta HTML debe ser estilizada individualmente. La gestión de tu sitio web puede resultar demasiado difícil si sólo utilizas CSS en línea.

Sin embargo, el CSS en línea en HTML puede ser útil en algunas situaciones. Por ejemplo, en los casos en los que no tienes acceso a archivos CSS o necesitas aplicar estilos para un solo elemento.
