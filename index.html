¿Que es HTML?

HTML (HyperText Markup Language) es un lenguaje de marcado de hipertexto que se utiliza para crear y estructurar páginas web:

  • Define el significado y la estructura del contenido de una página web.
  • Utiliza etiquetas y marcadores para definir cómo se presenta el contenido en un navegador.
  • Permite incluir texto, imágenes, videos, enlaces y otros recursos multimedia.

Para mostrar contenido en un navegador web, HTML utiliza etiquetas y elementos que permiten etiquetar texto, imágenes y otro contenido. Por ejemplo, para indicar un párrafo, se utiliza la etiqueta <p>.

HTML es la base para la construcción de páginas web, blogs, ecommerces, redes sociales y otros sitios accesibles desde un navegador.

Historia HTML

El origen de HTML se remonta a 1980, cuando el físico Tim Berners-Lee, investigador del CERN (Organización Europea para la Investigación Nuclear) propuso un nuevo sistema de “hipertexto” para compartir documentos. Los sistemas de hipertexto habían sido desarrollados años antes.

Tras finalizar el desarrollo de su sistema, Tim Berners-Lee lo presentó a una convocatoria organizada para desarrollar un sistema de hipertexto para internet. Después de unir sus fuerzas con el ingeniero de sistemas Robert Cailliau, presentaron la propuesta ganadora llamada WorldWideWeb (W3). El primer documento formal con la descripción de HTML se publicó en 1991 bajo el nombre “HTML Tags” (Etiquetas HTML) y todavía hoy puede ser consultado en línea a modo de reliquia informática.

Editores de HTML

Los editores de HTML son herramientas que permiten crear y editar código HTML para escribir la base de un sitio web. Algunos de los mejores editores de HTML son:

  • Brackets
  • Komodo
  • Bluefish
  • Notepad++
  • Setka
  • Visual Studio Code
  • Atom
  • Sublime Text
  • Apache NetBeans
  • CKEditor

Los editores de HTML suelen ofrecer funciones básicas como:

  • Resaltado de sintaxis
  • Autocompletado
  • Validación
  • Insertar elementos y estructuras HTML de uso común
  • Traducción del texto a otros lenguajes, como CSS, XML o JavaScript

Tu primer documento HTML

Para crear un documento HTML, puedes seguir estos pasos:

  1. Abrir un editor de texto, como el Bloc de Notas de Windows o el Procesador de textos de Ubuntu.
  2. Iniciar el documento con la declaración de tipo de documento <!DOCTYPE html >.
  3. Comenzar el documento con la etiqueta <html> y terminarlo con </html>.
  4. Entre <body> y </body> se encuentra la parte visible del documento HTML.
  5. Para crear párrafos de texto, se puede utilizar la etiqueta <p>.
  6. Para abrir un elemento, se coloca la etiqueta de apertura <em> al comienzo de la línea.
  7. Para cerrar el elemento, se coloca la etiqueta de cierre </em> al final de la línea.
  8. Guardar el documento.
  9. Abrir el documento HTML y verificar que se muestre el contenido en el navegador configurado por defecto.

¿Que son Etiquetas en HTML?

Las "tags" HTML, o "etiquetas" HTML, son códigos utilizados para "marcar" el texto de una página web, con el fin de dar instrucciones al navegador sobre cómo mostrarlo.

Es decir, las etiquetas HTML son el lenguaje utilizado para estructurar y definir el contenido en un documento HTML. Estas etiquetas se encuentran en el HTML (o Lenguaje de Marcado de Hipertexto) de cada página.

Cada etiqueta contiene instrucciones sencillas que indican al navegador cómo dar formato al texto y a definir los diversos elementos de la página web. Al aplicar estas etiquetas de marcado a los diferentes elementos del texto, se indica al navegador cómo mostrarlos al usuario, lo que permite crear páginas web estructuradas y con un diseño coherente.

Por ejemplo, puedes utilizar etiquetas HTML para aplicar cursivas, crear saltos de línea, insertar objetos multimedia, crear listas con viñetas o para definir diferentes tipos de contenido en una página web, como encabezados, párrafos, imágenes, enlaces, formularios y mucho más.

En resumen, las etiquetas HTML son esenciales para diseñar y organizar el contenido de una página web de manera efectiva y clara.

¿Que son Atributos en HTML?

Los atributos en HTML son fragmentos de código que se utilizan para modificar el comportamiento o la visualización de los elementos HTML. Se agregan a las etiquetas HTML utilizando la sintaxis "nombre de atributo=valor".

Algunos ejemplos de atributos HTML son:

  • accept: Lista de tipos de archivos que el servidor acepta
  • align: Especifica el alineamiento horizontal del elemento
  • alt: Texto alternativo en caso de que la imagen no se pueda mostrar
  • autocomplete: Indica si los controles en un formulario pueden tener por defecto sus valores completados automáticamente por el navegador
  • bgcolor: Color de fondo del elemento
  • color: Establece el color de texto
  • contenteditable: Indica si el contenido del elemento es editable
  • id: Define un identificador único (ID) que no debe repetirse en todo el documento
  • role: Describe la función de un elemento en programas que puedan hacer uso de él, tales como lectores de pantallas o magnificadores

Existen diferentes tipos de atributos, como los requeridos, opcionales, estándares y de evento.

Reglas fundamentales en HTML

Algunas reglas fundamentales para HTML son:

  1. Formato: Se deben usar dos espacios para la sangría y no debe haber espacios en blanco al final.
  2. Sintaxis: Se debe utilizar la sintaxis HTML5.
  3. Atributos: Todos los atributos deben estar entre comillas dobles.
  4. Etiquetas: Las etiquetas no distinguen entre mayúsculas y minúsculas, pero es recomendable escribirlas en minúsculas para mayor legibilidad.
  5. Atributos en etiquetas: Los atributos se incluyen en la etiqueta de apertura de un elemento, no en la de cierre.
  6. Estructura: La estructura HTML de una página web se compone de las etiquetas <html>, <head> y <body>.
  7. Etiqueta de apertura: Las etiquetas de apertura comienzan con el símbolo < >.
  8. Etiqueta de cierre: Las etiquetas de cierre terminan con el símbolo </ >.
  9. Contenido: El contenido de una etiqueta son las instrucciones sobre cómo mostrar el elemento en la página.

Etiquetas Para hacer un documento HTML

Para crear un documento HTML, se utilizan etiquetas que se escriben entre ángulos y que se componen de tres partes:

  1. Una etiqueta de apertura que comienza con el símbolo < >
  2. Contenido, que son las instrucciones para mostrar el elemento en la página
  3. Una etiqueta de cierre que termina con el símbolo </ >

Algunas etiquetas HTML que se pueden usar para crear un documento son:

  • <!DOCTYPE html>: Define el tipo de documento
  • <html>: Inicia el documento
  • <head>: Contiene información general del documento, como el título de la página, metaetiquetas, referencias a scripts y estilos
  • <title> </title>: Inserta el título del documento
  • <body> </body>: Señala el cuerpo del documento
  • <h1> ... <h6>: Muestra títulos y subtítulos
  • <div>: Define un bloque de contenido o sección de la página web
  • <a>: Realiza enlaces hacia documentos externos o internos
  • <strong>: Define texto importante en el documento
  • <br>: Realiza un salto de línea
  • <p>: Representa un párrafo
  • <img>: Coloca una imagen en el documento

Las etiquetas se pueden escribir en mayúsculas o minúsculas, pero las últimas especificaciones del W3C indican que deben ir en minúsculas para que el documento esté correctamente creado.
